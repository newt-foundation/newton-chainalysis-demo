### Newton Chainalysis Sanctions Policy Demo

##### Local Setup Instructions

```
$ git clone https://github.com/newt-foundation/newton-chainalysis-demo.git
$ cd newton-chainalysis-demo
```

Then replace the `ALCHEMY_URL` (in `/src/constants/alchemy.ts`) with your own Alchemy wss URL.

```
$ pnpm install
$ pnpm start
```
